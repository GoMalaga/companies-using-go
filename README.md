## Criteria for inclusion

A company should meet all of these to be listed:

- **Málaga presence** — an office or team in Málaga (city or province), or it actively hires Go developers based in Málaga.
- **Go in production** — Go is used in real products or infrastructure, not just experiments or side projects.
- **Verifiable** — ideally there is public evidence: job postings, engineering blog posts, conference/meetup talks, or open source repositories.

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for the details. In short: add a row to the table (keeping alphabetical order), one company per pull request. If you'd rather not open a PR, [file an issue](../../issues/new?template=add-company.yml) and we'll add it for you.

Corrections and removals (company closed, no longer uses Go, left Málaga) are just as welcome as additions.
