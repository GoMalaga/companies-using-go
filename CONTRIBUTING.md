# Contributing

Thanks for helping keep this list useful! Contributions of any size are welcome.

## Adding a company

1. Check the [criteria for inclusion](README.md#criteria-for-inclusion) and make sure the company isn't already listed.
2. Edit `README.md` and add a row to the table, **keeping alphabetical order** by company name:

   ```markdown
   | Acme | [acme.com](https://acme.com) | Payments backend, microservices | [Careers](https://acme.com/careers) |
   ```

   - **Company** — the public brand name.
   - **Website** — link to the main site.
   - **How they use Go** — one short line (services, tooling, infra…). No marketing copy.
   - **Jobs** — link to their careers page, or leave empty (`—`) if there isn't one.

3. Open a pull request with **one company per PR**. In the description, add any public evidence that they use Go in Málaga (job posting, blog post, talk, repo) — it speeds up review a lot.

Not comfortable with pull requests? [Open an issue](../../issues/new?template=add-company.yml) instead and we'll take care of it.

## Updating or removing a company

Companies change: they close offices, get acquired, or move away from Go. If you spot outdated information, open a PR or an issue explaining what changed. Removals are as valuable as additions — the list is only useful if it's accurate.

## Ground rules

- Work at one of these companies? Great — first-hand additions are the most reliable. Just keep the description factual.
- Don't list a company based on rumor alone; if there's no public evidence, say in the PR how you know.
- Be respectful in discussions. We're all here to make the Málaga Go community more visible.
